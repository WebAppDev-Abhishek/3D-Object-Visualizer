üöÄ 3D Scene Explorer

A versatile web-based 3D scene explorer built with React and Three.js, allowing users to interactively create, manipulate, and visualize 3D objects and lights in a dynamic environment.

‚ú® Features

  • Add Basic Shapes: Easily add cubes, spheres, cylinders, cones, and tori to the scene.
  • Add Point Lights: Introduce dynamic lighting to your scene with adjustable color and intensity.
  • Interactive Object Manipulation: Select objects (shapes and lights) and use TransformControls to move, rotate, and scale them directly in the 3D space.
  • Visual Selection Highlight: Selected objects change to a lighter color for clear visual feedback.
  • Property Editing: Adjust the color and scale of selected shapes, and the color and intensity of selected lights via intuitive UI controls.
  • Camera Controls:
    • Orbit camera around the scene using mouse/trackpad.
    • Move camera using arrow keys (forward, backward, strafe left, strafe right).
    • Toggle camera motion (including auto-rotation and arrow key movement) on/off by pressing the Spacebar.
    • Toggle auto-rotation of the scene.
  • Scene Management:
    • Undo and Redo actions for scene modifications (adding, removing, transforming objects).
    • Remove the last added object or the currently selected object.
    • Reset the entire scene to its initial empty state.
  • Environment Customization:
    • Toggle the visibility of the grid helper.
    • Change the background color of the 3D scene.
  • Image Import: Import local image files and display them as textured planes in the 3D scene.
  • Responsive Design: The application adapts to different screen sizes.

üöÄ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

Prerequisites

You need to have Node.js and npm (Node Package Manager) installed on your system.

  • Node.js (includes npm)

Installation

1. Clone the repository (or create a new React app):

If you're starting from scratch, you can create a new React app and then copy the source code.

# Option 1: Create a new React app (recommended for fresh start)
npx create-react-app my-3d-scene-explorer
cd my-3d-scene-explorer

# Option 2: If you have an existing project, navigate to its root
# cd path/to/your/project

2. Install project dependencies:

npm install three
# For Tailwind CSS (if not already set up by your React app template):
npm install -D tailwindcss postcss autoprefixer

3. Set up Tailwind CSS (if using a new React app or not already configured):

Initialize Tailwind CSS:

npx tailwindcss init -p

This will create tailwind.config.js and postcss.config.js.

Update tailwind.config.js to scan your source files:

Running the Application

Once everything is set up, you can start the development server:

npm start
# or if using Vite:
# npm run dev

This will open the application in your browser, usually at http://localhost:3000 (or http://localhost:5173 for Vite).

üí° Usage

  • Add Objects: Use the buttons in the header to add various 3D shapes and point lights to the scene. You can set their initial X, Y, Z positions.
  • Select Objects: Click on any object (shape or light helper) in the 3D view to select it. The selected object will turn a lighter shade.
  • Manipulate Objects: Once selected, use the TransformControls (the colored axes that appear) to move, rotate, or scale the object.
  • Edit Properties: When an object is selected, use the "Selected" panel in the header to change its color, scale (for shapes), or intensity (for lights).
  • Camera Navigation:
    • Mouse/Trackpad: Click and drag to orbit the camera. Scroll to zoom.
    • Arrow Keys: Use ArrowUp, ArrowDown, ArrowLeft, ArrowRight to move the camera.
    • Spacebar: Press the Spacebar to toggle camera movement (including OrbitControls and arrow key movement) on/off.
  • Scene Controls:
    • Grid On/Off: Toggle the visibility of the ground grid.
    • Undo/Redo: Revert or reapply recent scene changes.
    • Remove Last/Selected: Delete objects from the scene.
    • Reset Scene: Clear all objects and reset the scene to its default state.
    • Background: Change the background color of the scene.
    • Auto-rotate: Toggle automatic rotation of the camera around the scene.
  • Import Images: Use the "Import Image" button to load a local image file as a textured plane in the scene.
